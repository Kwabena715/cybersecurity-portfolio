# MITRE ATT&CK Mapping

## ATT&CK Tactic

TA0007 – Discovery

---

## ATT&CK Technique

T1046 – Network Service Discovery

---

## Detection Method

Network scanning using Nmap.

---

## Defensive Purpose

This project demonstrates how defenders can use network discovery to:

- Identify assets.
- Validate exposed services.
- Detect unauthorized devices.
- Establish a baseline for future investigations.

---

## Findings

No attacker techniques were observed.

The scan identified only legitimate Windows networking services.

No Indicators of Compromise were detected.
